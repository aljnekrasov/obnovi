# Настройка Google Sheets для приёма заявок

## Шаг 1. Создать таблицу
1. Перейдите на [sheets.google.com](https://sheets.google.com)
2. Создайте новую таблицу (или используйте существующую)

## Шаг 2. Добавить скрипт
1. В таблице: **Расширения** → **Apps Script**
2. Удалите весь код в редакторе
3. Скопируйте содержимое файла `Code.gs` и вставьте в редактор
4. Сохраните (Ctrl+S)

## Шаг 3. Развернуть веб-приложение
1. Нажмите **Развернуть** → **Новое развёртывание**
2. Тип: выберите **Веб-приложение**
3. Укажите:
   - **Выполнять от имени**: я
   - **У кого есть доступ**: все пользователи
4. Нажмите **Развернуть**
5. **Скопируйте URL веб-приложения** (например, `https://script.google.com/macros/s/.../exec`)

## Шаг 4. Вставить URL в сайт
Замените `YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL` на скопированный URL в трёх файлах:
- `index.html` (переменная GS_WEBAPP_URL)
- `b2b.html` (переменная GS_WEBAPP_URL)
- `projects.html` (переменная GS_WEBAPP_URL)

Пример:
```javascript
const GS_WEBAPP_URL = 'https://script.google.com/macros/s/AKfycbxxx.../exec';
```

Готово. Заявки с квиза, формы партнёров и формы проектов будут сохраняться в таблицу.
